# Filter

Java 25.0.2 Проверка версии: java -version

Maven 3.9.11 Проверка версии: mvn -version

Сторонние библиотеки: https://mvnrepository.com/artifact/info.picocli/picocli/4.7.7

Сборка проекта
Откройте терминал (командную строку) в корневой директории проекта (там, где находится файл pom.xml).
Выполните команду сборки: mvn clean package
После успешной сборки в папке будет создан исполняемый JAR-файл.

Формат входных файлов
Входные файлы могут содержать смесь целых чисел, чисел с плавающей точкой и строк.

Каждый элемент (число или строка) должен располагаться на отдельной строке. Пустые строки cчитаются строками нулевой длины и записываются в файл со строками.

Целые числа: Последовательности цифр, возможно, начинающиеся с символа - (например, 45, -100, 1234567890123456789).

Числа с плавающей точкой: Числа, содержащие десятичную точку, или записанные в экспоненциальном формате (например, 3.1415, -0.001, 1.528535047E-25).

Строки: Все остальное, что не может быть интерпретировано как целое или вещественное число (например, Lorem ipsum, Пример, Long).

Запуск утилиты
Утилита запускается из командной строки с указанием списка входных файлов и опциональных параметров
Пример: java -jar uXl.jar -s -a -p sample- in1.txt in2.txt

Опции командной строки:
-o <путь>	Задает путь для сохранения результирующих файлов. Если путь не указан, файлы создаются в текущей рабочей папке.	-o /home/user/results
-p <префикс>	Задает префикс для имен результирующих файлов. К префиксу будут добавлены стандартные имена: integers.txt, floats.txt, strings.txt.	-p sample-
-a	Включает режим добавления в существующие файлы. Если опция не указана, файлы результатов будут перезаписаны при каждом запуске.	-a
-s	Включает вывод краткой статистики в консоль (только количество записанных элементов для каждого типа).	-s
-f	Включает вывод полной статистики в консоль (количество, min, max, сумма, среднее для чисел; количество, длина самой короткой и самой длинной строки для строк).	-f
Важно: Опции -s и -f являются взаимоисключающими. Если не указать ни одну из них, статистика выводиться не будет. Если указать обе, приоритет будет у опции -f.
